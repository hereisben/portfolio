# Portfolio Website

This is my personal portfolio website.

The goal of this project is to present my background, projects, skills, and writing in a clean and modern way.  
The design is inspired by modern developer portfolios with a dark theme, smooth animations, and clear structure.

---

## Tech Stack

- **Next.js** – React framework for routing, SEO, and performance
- **React** – Component-based UI
- **Tailwind CSS** – Styling and layout
- **Framer Motion** – Animations and transitions
- **shadcn/ui (Radix UI)** – Accessible UI components
- **Lucide Icons** – Icons
- **MDX** – Blog / journal content
- **Vercel** – Deployment

---

## Features (Planned)

- Hero section with introduction and call-to-action
- About / Journey timeline
- Featured projects
- Skills and tools section
- Blog / Journal using MDX
- FAQ section
- Contact form
- Responsive design (desktop + mobile)
- Smooth animations on scroll and hover

---

## Project Structure (Planned)

```txt
src/
├─ app/            # Next.js App Router
├─ components/     # Reusable UI components
├─ sections/       # Page sections (Hero, Projects, etc.)
├─ content/        # MDX blog posts
├─ lib/            # Utilities and helpers
├─ styles/         # Global styles
```
